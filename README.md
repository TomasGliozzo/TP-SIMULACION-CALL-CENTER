# TP-SIMULACION-CALL-CENTER
Este estudio optimiza un Call Center mediante simulación, proponiendo un modelo híbrido (IA + capacitación) para mantener un SLA del ~90%. La IA absorbe interacciones reduciendo operadores junior hasta un 77%, mientras la capacitación agiliza resoluciones con ROI positivo. Esta sinergia minimiza costos y maximiza la resiliencia.
